# 项目名称
这是一个用于管理电影的 Next.js 应用程序，旨在提供一个用户友好的界面来浏览、搜索和管理电影信息。

## 使用方法

1. 克隆这个仓库：
   ```bash
   git clone https://github.com/1998SUIYUE/av_manager_nextjs.git
   ```

2. 进入项目目录：
   ```bash
   cd av_manager_nextjs
   ```

3. 安装依赖：
   ```bash
   npm install
   ```

4. 运行开发服务器：
   ```bash
   npm run dev
   ```

5. 打开浏览器访问 [http://localhost:3000](http://localhost:3000) 查看应用。
